# Shopping Cart Fun 🛒

A simplified shopping cart application designed to demonstrate core e-commerce functionalities. This project allows users to view products, add them to a cart, and manage their shopping selection.

## 🚀 Features

* **Product Listing:** Browse a variety of available products with images and prices.
* **Add to Cart:** One-click functionality to add items to the shopping cart.
* **Cart Management:**
    * Increase or decrease the quantity of items.
    * Remove items from the cart.
    * **Dynamic Pricing:** Automatically calculates subtotal and total prices.
* **Responsive Design:** Works on desktop and mobile devices.

## 🛠️ Technologies Used

* **Frontend:** [HTML5, CSS3, JavaScript (ES6+)] *OR* [React.js / Vue.js]
* **Styling:** [Bootstrap / Tailwind CSS / Custom CSS]
* **Icons:** [FontAwesome / React Icons]
* **Version Control:** Git & GitHub

🔧 Installation & Setup

Bash

Bash

Bash

💻 Usage
Open the application.

Click on "Add to Cart" buttons for various products.

Click the Cart Icon to view your selected items.

Adjust quantities or remove items to see the total price update in real-time.
